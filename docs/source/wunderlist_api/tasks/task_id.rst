/<task_id>
=============

Modify or delete tasks.

PUT
---
Modify a task.

Parameters
""""""""""
:task_id: Passed in the URL.
:note: The new note for the task.
:due_date: The new due date for the task (ISO format as usual).
:recurrence_count: Not positive.
